# Todo App

The Todo App is a simple application that allows you to manage your tasks and stay organized.

## Features

- Create new tasks with a title and optional description
- Mark tasks as complete or incomplete
- Edit existing tasks
- Delete tasks
- Filter tasks by status (complete/incomplete)
- Clear all completed tasks

## Prerequisites

- Node.js (version 18.16.0)
- npm (version 9.4.0)
- react-native (version 0.72.0)
- expo (version 6.3.8) (Expo is an open-source platform for making universal native apps for Android, iOS, and the web with JavaScript and React.)

## Installation

1. Clone the repository:

   ```git clone https://github.com/your-username/todo-app.git```

2. Navigate to the project directory:

   ```cd todo-app```

3. Install the dependencies:

   ```npm install```

## Usage

1. Start the application:

   ```npx expo start```

2. Scan the QR code with the Expo Go app
